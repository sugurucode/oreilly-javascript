## TypeScriptとは
ypeScript is JavaScript with syntax for types.
https://www.typescriptlang.org/ja/

## Flowとは
Flow is a static type checker for JavaScript.
https://flow.org/

## どちらが主流？
Typescriptが主流。
[Github octoverse 2025](
https://github.blog/news-insights/octoverse/octoverse-a-new-developer-joins-github-every-second-as-ai-leads-typescript-to-1/#the-top-programming-languages-of-2025-typescript-jumps-to-1-while-python-takes-2)のランキングによると、Githubで最も利用されているらしい。
対してFlowはTop10にも入っていない。

AIがコード生成するには学習データが必要。typescriptは最も利用されているので、最も学習データがある。学習データが多いということは、高品質なコード生成が可能
→AIでのコーディングが主流の現代に最もあうから人気？
