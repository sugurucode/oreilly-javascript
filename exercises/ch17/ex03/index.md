## npxを利用する主な利点

- **グローバル環境をクリーンに保てる**
  `npm install -g` が不要になるため、PCの環境を汚さず、プロジェクト間でのバージョン競合を防ぐ。
- **常に最新バージョンを実行できる**
  ローカルにないパッケージは実行時に一時ダウンロードされるため、常に最新版を利用できる（`npx package@version` のようにバージョン指定も可能）。
- **ローカルパッケージの実行が簡単になる**
  プロジェクトにインストールしたツール（Jestなど）を、長いパス（`./node_modules/.bin/jest`）を書かずに、直接 `npx jest` と短く実行できる。
