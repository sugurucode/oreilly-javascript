# localStorage と sessionStorage それぞれに保存されたデータの有効期限がどのように異なるか、実際に動作確認して結果を記述しなさい。

※ 有効期限：どのタイミングで消えるかという定義で調べました。

## 検証パターン

- ページリロード
- タブ削除→タブ復元(ctrl+shift+t)
- ブラウザ再起動

### localstorage

- ページリロード
  データは残る
- タブ削除→タブ復元(ctrl+shift+t)
  データは残る
- タブ削除→タブ復元（新規タブにURL直接指定）
  データは残る
- ブラウザ再起動（新規タブにURL直接指定）
  データ残る

### sessionstorage

- ページリロード
  データは残る
- タブ削除→タブ復元(ctrl+shift+t)
  データは残る
- タブ削除→タブ復元（新規タブにURL直接指定）
  データ消える
- ブラウザ再起動（新規タブにURL直接指定）
  データ消える

## 検証結果の比較表

| 検証パターン                   | localStorage | sessionStorage |
| :----------------------------- | :----------: | :------------: |
| ページリロード                 |      〇      |       〇       |
| タブ削除 → 復元 (Ctrl+Shift+T) |      〇      |       〇       |
| タブ削除 → 新規タブ (URL入力)  |      〇      |       ×        |
| ブラウザ再起動 (新規タブ)      |      〇      |       ×        |
